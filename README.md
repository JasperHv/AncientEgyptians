## Introduction
Ancient Egyptians Game is a strategic card game inspired by ancient Egyptian mythology. Originally developed as part of a 
university software design course, this project showcases advanced design patterns and best practices in software engineering. 
The game involves players competing to build the most powerful deck of cards, each representing different deities, artifacts, 
and mythical creatures from ancient Egypt. Players take turns drawing cards, playing them strategically to gain advantages, 
and attempting to outmaneuver their opponents. The project is currently being refined beyond the initial coursework version.
## Installation
To set up the project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/yourproject.git
   ```

2. Navigate to the project directory:
   ```bash
   cd yourproject
   ```

3. Build the project using Gradle:
   ```bash
   ./gradlew build
   ```

## Usage
To run the game, open the `AncientEgyptiansApp.java` file located at `src/main/java/nl/vu/cs/ancientegyptiansgame/AncientEgyptiansApp.java` and hit the "Run File" button in your IDE.
